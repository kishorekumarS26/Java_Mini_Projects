# 🎵 Java Music Player

A simple console-based music player developed using Java. The application plays a WAV audio file and provides basic playback controls through the terminal. It demonstrates the use of Java's Sound API, file handling, and exception handling.

---

## Features

- Play a WAV audio file
- Pause and resume playback
- Reset the audio to the beginning
- Quit the application
- User-friendly console menu

---

## Technologies Used

- Java
- Java Sound API
- File Handling
- Exception Handling
- Scanner for User Input

---

MusicPlayer/

---

## How to Run

1. Place a valid `.wav` audio file in the project directory.
2. Update the file path if necessary.
3. Compile and run:

```bash
javac MusicPlayer.java
java MusicPlayer
```

---

## Controls

| Key | Action |
|------|--------|
| P | Play |
| S | Stop |
| R | Reset |
| Q | Quit |
