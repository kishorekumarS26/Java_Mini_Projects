# ⏳ Java Countdown Timer

A console-based countdown timer built using Java. The application counts down from a user-specified number of seconds and displays the remaining time every second using Java's Timer and TimerTask classes.

---

## Features

- User-defined countdown
- Real-time countdown display
- Automatic completion message
- Uses Java Timer API

---

## Technologies Used

- Java
- Timer
- TimerTask
- Scanner

Compile and run:
