# Java Alarm Clock

A console-based alarm clock application developed using Java.

## Features

- Set alarm in HH:mm:ss format
- Displays current time
- Plays a WAV audio file
- Uses multithreading
- Stop alarm by pressing Enter

## Technologies

- Java
- LocalTime API
- Runnable Interface
- Java Sound API

## Files

- Main.java
- AlarmClock.java
- AudioFile.wav
