#  Java Compound Interest Calculator

A console-based Java application that calculates the final amount earned through compound interest based on the principal amount, annual interest rate, compounding frequency, and investment duration.

---

## Features

- User-friendly console interface
- Calculates compound interest
- Accepts customizable inputs
- Displays result with two decimal places

---

## Technologies Used

- Java
- Math API
- Scanner
- Mathematical Computation

## Formula Used

```
A = P × (1 + r / n)^(n × t)
```

Where:

- **P** = Principal Amount
- **r** = Annual Interest Rate
- **n** = Number of Times Compounded Per Year
- **t** = Number of Years
- **A** = Final Amount

---

## How to Run

Compile and execute:

```bash
javac CompoundInterestCalculator.java
java CompoundInterestCalculator
```
