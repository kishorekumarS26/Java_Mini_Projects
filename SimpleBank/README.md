# 🏦 Java Simple Bank

A console-based banking application developed in Java that simulates basic banking operations such as checking account balance, depositing money, and withdrawing funds.

---

## Features

- View account balance
- Deposit money
- Withdraw money
- Input validation
- Menu-driven interface

---

## Technologies Used

- Java
- Object-Oriented Programming
- Methods
- Scanner
- Conditional Statements

---

## Project Structure

```
SimpleBank/
│
├── SimpleBank.java
└── README.md
```

---

## How to Run

Compile and execute the program:

```bash
javac SimpleBank.java
java SimpleBank
```

---

## Available Operations

- Show Balance
- Deposit Money
- Withdraw Money
- Exit
