# Java Mini Projects

A collection of beginner-friendly Java console applications developed while learning Java programming. These projects demonstrate fundamental programming concepts, object-oriented programming, multithreading, file handling, timers, and Java's built-in APIs.

## Projects

| Project | Description |
|---------|-------------|
| Alarm Clock | Console-based alarm clock with audio notification |
| Music Player | WAV audio player with play, stop, reset, and quit controls |
| Simple Bank | Banking simulation with deposit, withdrawal, and balance management |
| Countdown Timer | Countdown timer using Java Timer and TimerTask |
| Compound Interest Calculator | Calculates compound interest based on user inputs |

## Technologies

- Java
- Object-Oriented Programming
- Java Time API
- Java Sound API
- Multithreading
- Exception Handling


## How to Run the Projects

### Prerequisites

- Java Development Kit (JDK 8 or above)
- Any Java IDE (IntelliJ IDEA, Eclipse, VS Code) or Command Prompt/Terminal

### Clone the Repository

```bash
git clone https://github.com/kishorekumarS26/Java_Mini_Projects.git
```

Navigate to the project folder:

```bash
cd Java_Mini_Projects
```

### Compile and Run

Each project is independent. Open the desired project folder and compile the Java file(s).

#### Alarm Clock

```bash
cd AlarmClock
javac Main.java AlarmClock.java
java Main
```

> **Note:** Ensure the `AudioFile.wav` audio file is present in the specified path before running the project.

---

#### Music Player

```bash
cd MusicPlayer
javac MusicPlayer.java
java MusicPlayer
```

> **Note:** Ensure the `AudioFile.wav` file is available in the project directory.
